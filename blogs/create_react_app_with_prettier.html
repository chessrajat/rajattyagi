Setup Reactjs with vscode and prettier

13 Feb 2021
Reactjs

Hey everyone , so lets start setting up a new React project in vscode and with prettier extension.
Lets first create the react project with create-react-app . I am going to name my project reactdesigns

npx create-react-app reactdesigns
cd reactdesigns
code .

First it will create the react app with utility create-react-app for more on this check out this here
Then we just move into the newly created app and open it in vs code.

Now You have a working react app in vscode , you can open the integrated terminal in the vs code and run npm start to get your working project open in the browser.

Setting up Prettier

  • Install the extension prettier from the extensions in vscode.

  • Installing dev dependencies for prettier.

  • npm i -D eslint-plugin-prettier prettier

  • After this you should see these two in package.json file

  • Now create a file in the root directory named .eslintrc

      {
      "extends": "react-app",
  	"plugins": ["prettier"],
  	"rules": {
  		"prettier/prettier":["error",{
  		"endOfLine":"auto"
  		}]
  	}
  }

Put these above settings in the .eslintrc file.
Now just needs to create vscode settings so our code will auto format on every save

  • Create a folder .vscode in the root directory

  • Inside it create a file settings.json

      {
  	"editor.formatOnSave": true,
  	"editor.tabSize": 2,
  	"editor.defaultFormatter": "esbenp.prettier-vscode"
  }

You can change the tabsize as you prefer.
So that’s it. Now your react project is ready with auto formatting every time your press ctrl+s
