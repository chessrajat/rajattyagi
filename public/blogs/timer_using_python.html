Hey , everyone so. How to build a Timer in python?
we will first start by setting up our python virtual environment

  • create a directoryTimer
  • python -m venv venv

create a file Main.py

Before starting you need to install a package playsound 

pip install playsound

So Now lets start with our function our_timer, it takes seconds as a arguments 

def  our_timer(seconds):
	thread =  Thread(target=playsound,args=("./Timer/beep_sound.mp3",), daemon=True)
	for i in  range(seconds-1, -1, -1):
		sys.stdout.write("\r Timer : %s"  %i)
		if i ==  9:
			thread.start()
		time.sleep(1)
		sys.stdout.flush()

Here we are using thread to run the sound on different thread so , when the sound is running it won’t interrupt our main timer thread.
We are also using sys.stdout.flush() to put the new time in the same place as the old one. instead of printing every second in the new line.

we can call the function like: our_thread(20) so it will run for 20 seconds.

we are calling the function directly lets build a main function that will take argument which function to run the timer 

if  __name__=="__main__":

    seconds =  int(sys.argv[1])
    our_timer(seconds)

Here we are taking one argument as integer .

Example :

  • python Main.py 30
  • python Main.py 50
